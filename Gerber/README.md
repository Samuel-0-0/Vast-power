## 焊接说明

1、收到PCB后，先将2块小的附件板切割下来，注意请不要硬掰。

2、下载此焊接辅助文件 [PCB_vast-power_2022-12-20](PCB_vast-power_2022-12-20.html)

3、建议新手分区域焊接测试，先焊接①部分后测试固件是否能正常写入，然后焊接②部分测试继电器是否正常，最后焊接其他部分测试。
